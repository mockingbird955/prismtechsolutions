# Prism Tech Solutions Website

Starter scaffold for the Prism Tech Solutions business website.

## Structure

- `index.html` - Main landing page
- `css/styles.css` - Base styles
- `js/main.js` - Base JavaScript file
- `assets/` - Static assets (images, icons, etc.)

## Run locally

Open `index.html` directly in a browser, or use a local server later if needed.
